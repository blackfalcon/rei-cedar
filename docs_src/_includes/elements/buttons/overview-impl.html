When constructing a button you can select modifiers for state, size, and shape.

Type Name Default Description
state btn-primary true
state btn-default true
state btn-contrast true

btn-contrast is used in conjunction with btn-primary to achive the black contrast button.

btn-contrast is used in conjunction with .btn-default to achive the white contrast button.

size btn-lg true btn-lg is the default size value and is inherited into the btn element.
size btn-sm false btn-sm is an additive class on button and is used in conjunction with inline form fields or when repeated in a list such as "Register" on a classes and Events page.
shape btn-xs false btn-xs is an additive class on button and is intended for use in very limited situations where space is at an extreme premium.
shape btn-fixed false btn-fixed This is an additive class that prevents the button from going full-width at the xs breakpoint.
