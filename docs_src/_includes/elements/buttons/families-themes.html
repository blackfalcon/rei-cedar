Core buttons

CORE buttons are meant for transactional use. They can be considered the "utilitarian" buttons, seen within product focused areas of the site, like Search Results, Product Details, or login pages. These buttons help the user act on an informational, research or purchase path of an REI product.

Button examples: "Submit", "Checkout", "Add to cart", "See more", "Continue shopping".

Use for a single transactional CTA; when only one action is needed.

When grouped, it is used for the primary action of the grouping.
